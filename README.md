# Spam and Ham Data in Bahasa Indonesia

This repository contains data used in our publication entitled 'Indonesian language email spam detection using N-gram and Naïve Bayes algorithm'.

The spam and ham email samples were collected from 30 people, where each of them provided 10 spam and 10 ham samples.
In total, there are 300 spam and 300 ham email samples, which are further split into training (400 samples) and test sets (200 samples).

*RELATED ARTICLE*

For more detailed information related to the data used and the application, please refer to the following publication:
Vernanda, Y., Hansun, S., Kristanda, M. B. Indonesian language email spam detection using N-gram and Naïve Bayes algorithm. Bulletin of Electrical Engineering and Informatics (BEEI). 9:5, 2012-2019 (2020). https://doi.org/10.11591/eei.v9i5.2444
